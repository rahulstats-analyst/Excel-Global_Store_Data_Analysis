## 🎯 Project Objective
In the highly competitive global retail market, understanding sales trends across different regions, categories, and customer segments is crucial. This project leverages Microsoft Excel to clean, process, and analyze a massive global store dataset. The resulting interactive dashboard provides business leaders with the insights needed to identify profitable regions, optimize product inventory, and improve overall sales strategies.

---

## 🛠️ Key Skills & Techniques Applied

* 🧹 **Data Cleaning:** Handled missing values, standardized text formats, and removed duplicates to ensure data accuracy.
* 🧮 **Advanced Formulas:** Utilized functions like `VLOOKUP`, `INDEX/MATCH`, `IF/IFS`, and date functions to engineer new analytical features.
* 📊 **PivotTables & PivotCharts:** Summarized millions of data points into digestible, cross-tabulated metrics.
* 🎛️ **Interactive Dashboarding:** Built a user-friendly interface utilizing **Slicers** and **Timelines** to allow for dynamic, on-the-fly data filtering.
* 📉 **Trend Analysis:** Visualized Year-over-Year (YoY) and Quarter-over-Quarter (QoQ) growth patterns.

## 🚀 How to Explore the Data

1. **Download:** Click on the `Global Store Data Analysis.xlsx` file in this repository and download it to your local machine.
2. **Open:** Open the file using Microsoft Excel (Office 365 or 2019+ recommended for full compatibility with slicers).
3. **Navigate:** Use the designated "Dashboard" tab to interact with the data.
4. **Interact:** Click the Slicers (e.g., Year, Region, Segment) to watch the PivotCharts update automatically!
